STILL US - LIVING LOVE LETTER
=============================

INSTRUCTIONS FOR PERSONALIZATION:

1. REPLACE PLACEHOLDER TEXT:
   - In apology section: "[be specific about your mistake]"
   - In memory cards: Add your actual shared memories
   - In reasons: Personalize each reason for her specifically

2. ADD YOUR ASSETS:
   - Record a voice note: Save as "assets/voice-note.mp3"
   - Add background music: Save as "assets/bg-music.mp3"
   - Add your photos to memory cards if desired

3. UPDATE EMAIL:
   - In script.js, line ~201: Replace "YOUR_EMAIL_HERE" with your actual email

4. CUSTOMIZE DISTANCE:
   - In script.js, line ~141: Change "824" to your actual distance in miles

5. DEPLOYMENT:
   - Upload ALL files to any web hosting (Vercel, Netlify, GitHub Pages)
   - For password protection (recommended):
     * Netlify: Add _redirects file with password protection
     * Vercel: Use password protection in project settings
     * Or use .htaccess for Apache servers

6. BEFORE SENDING:
   - Test on mobile (most important!)
   - Test voice note playback
   - Test email link works
   - Make sure it feels emotionally authentic

EMOTIONAL CHECKLIST:
☐ Does it feel honest, not performative?
☐ Is her autonomy respected?
☐ Are there no pressure tactics?
☐ Does it reflect YOUR genuine feelings?
☐ Would you feel safe receiving this?

TECHNICAL NOTES:
- All files are self-contained
- No external dependencies except fonts
- Works offline after first load
- Responsive on all devices
- Lightweight (fast loading)

Remember: This is a gift, not a demand.
Send it with an open hand and an open heart.
